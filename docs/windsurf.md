## 🛠️ Development Workflows

Windsurf workflows are available for automating common development and documentation tasks:

### 📝 Documentation Workflows
- **changelog-append**: Append recent chat session changes to changelog
- **api-reference-update**: Update API reference documentation
- **design-decision-document**: Document design decisions from discussions
- **roadmap-add-task**: Add new tasks to the project roadmap
- **documentation-health-check**: Validate documentation completeness
- **test-documentation-sync**: Keep testing docs in sync with code

### 🚀 Usage
```bash
# List available workflows
windsurf workflows list

# Run a specific workflow
windsurf workflows run changelog-append

# Validate workflow files
windsurf workflows validate

# Show workflow help
windsurf workflows help [workflow-name]
```

For detailed workflow documentation, see [`README.md`](../.windsurf/README.md).
